</html>

3. 글자태그
	1. 제목태그
<h1></h1> 은 제목태그 h1~h6까지잇고 h1이 글자크기가 가장큼

	2. 본문태그
(1) <p></p> 본문 글자 태그
(2) <br> 줄바꿈 태그
(3) <hr> 수평줄 태그

	3. a태그(앵커태그)
하이퍼링크 태그
<a href ="http://www.naver.com">네이버</a>
<a href ="#" 현재 페이지의 가장 첫번째 부분으로 이동</a>   처음으로가기?
<a href ="#alpha">id 속성 값이 alpha인 태그 쪽으로 이동</a>  해당 태그가 있는 위치로 이동 
						        ex)<h1 id="alpha">제목</h1> 일시 h1으로 이동
	4. 그외
<b></b>는 굵은 글자
<i></i>는 이탤릭체 요새 i태그는 아이콘을 의미하는 경우가 많다

루비문자 한자위에 표시되는 글자
<ruby>
        <span>大韓民國</span>  
        <rp></rp>   --> 예외처리
        <rt>대한민국</rt>
        <rp></rp>	--> 예외처리
</ruby>

사용자정의태그 : 존재하지않아도 동작은됨
<leehyeonoh>이현오태그</leehyeonoh> 

4. 목록태그
네비게이션 메뉴에 주로 사용된다
네비게이션 메뉴 : 페이지 이동에 사용되는 메뉴
    <h1>ol tag</h1>
    <ol>    --> 순서가 있는 목록 태그 앞에 1. 2. 3.이 붙는다
        <li>FaceBook</li>
        <li>Tweeter</li>
        <li>Linked in</li>
    </ol>
    
    <h1>ul tag</h1>
    <ul>  --> 순서가 없는 목록 태그 앞에 ●이 붙는다
        <li>FaceBook</li>
        <li>Tweeter</li>
        <li>Linked in</li>
    </ul>

중첩해서 사용가능하다
     <ul>
        <li>HTML5	--> ●
            <ol>
                <li>ㅎㅇㅎㅇ</li>  --> 1,2,3
                <li>ㅎ22</li>
                <li>ㅎ32</li>
            </ol>
        </li>
        <li>CSS3    --> ●
            <ul>
                <li>Animation</li>    --> ul중첩하니 ○이게나옴
                <li>3D Transform</li>
            </ul>
        </li>
    </ul>

<dl></dl> : 정의 목록 태그
<dt></dt> : 정의 용어 태그
<dd></dd> : 정의 설명 태그

5. 테이블 태그
<table border="1">            --> 숫자에따라 테이블 디자인이 바뀜
        <tr>                      --> tr : 행늘리기  행태그
            <th>Header1</th>  --> th : 열 늘리기 제목셀 태그
            <th>Header1</th>  
        </tr>
        <tr>
            <td>Data 1</td>	 --> td 일반셀 태그
            <td>Data 1</td>
        </tr>
        <tr>
            <td>Data 2</td>
            <td>Data 2</td>
        </tr> 
</table>

테이블 내부의 태그들
<table border="1">  --> 테이블 디자인
        <caption>테이블명</caption>  --> 테이블명(캡션)
        <colgroup>    --> 테이블안의 칼럼을 묶는다
            <col span="2" style="background: red" />   --> span 2 : 모든행 2열까지 여기선 모든행 2열까지 바탕색을 빨강으로
            <col style="background: blue"/>	      --> span 다음 칼럼의 행 배경색을 파랑으로한다
        </colgroup>

        <thead style="background: green">	--> 테이블 헤더색을 초록으로
            <tr>
                <th>테이블 헤더</th>
                <th>테이블 헤더</th>
                <th>테이블 헤더</th>
            </tr>
        </thead>

        <tbody>
            생략
        </tbody>

        <tfoot style="background: yellow">  --> 마지막 열 색을 노랑으로 바꾼다
            <tr>
                <td>테이블데이터 4</td>
                <td>테이블데이터 4</td>
                <td>테이블데이터 4</td>
            </tr>
        </tfoot>
    </table>



<th colspan="3">테이블 데이터</th> --> 셀의 너비 지정(가로)
<th rowspan="">테이블 데이터</th> -->  셀의 높이 지정 (세로)
